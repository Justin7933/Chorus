# Chorus Overview<p>
Chorus is a forum based website designed to bring people together and allow them to easily compile their ideas and thoughts into a concise thread.<p>
## Tools used 
  <ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
